## Deskripsi

Tugas ini merupakan implementasi materi **perulangan (looping) dalam Java** yang meliputi penggunaan `for`, `while`, dan operator modulus `%`.
Project terdiri dari 3 class sesuai soal untuk NIM Genap.

---

## Struktur Program

- **Soal1Genap.java**  
  Menggunakan perulangan `for` untuk menampilkan teks  
  "Saya senang belajar bahasa Java" sebanyak N kali sesuai input pengguna.

- **Soal2Genap.java**  
  Menggunakan perulangan `for` dan kondisi `if` untuk:
  - Menampilkan bilangan genap dalam rentang batasAwal sampai batasAkhir
  - Menampilkan proses perhitungan
  - Menampilkan jumlah deret bilangan genap

- **Soal3Genap.java**  
  Menggunakan perulangan `while` dan algoritma Euclidean (modulus) untuk menghitung Pembagi Bersama Terbesar (PBB) dari dua bilangan.

## Kesimpulan

Melalui tugas ini, saya memahami:
- Penggunaan perulangan `for`
- Penggunaan perulangan `while`
- Penerapan kondisi dalam loop
- Penggunaan operator modulus dalam perhitungan matematika
